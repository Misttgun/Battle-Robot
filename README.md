## Configuration des touches
- ZQSD pour se déplacer
- Espace pour sauter
- Clic de la souris gauche pour tirer
- F pour ramasser
- G pour reposer l'item actif
- 1,2,3,4,5 pour sélectionner les objets dans l'inventaire
